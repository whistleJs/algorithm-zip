# 두 수의 합

> https://school.programmers.co.kr/learn/courses/30/lessons/181846

### 문제 설명

0 이상의 두 정수가 문자열 `a`, `b`로 주어질 때, `a + b`의 값을 문자열로 `return` 하는 `solution` 함수를 작성해 주세요.

### 제한사항

- 1 ≤ `a`의 길이 ≤ 100,000
- 1 ≤ `b`의 길이 ≤ 100,000
- `a`와 `b`는 숫자로만 이루어져 있습니다.
- `a`와 `b`는 정수 0이 아니라면 0으로 시작하지 않습니다.
